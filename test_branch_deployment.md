# Test分支部署说明 - 留言回复功能

## 概述
在test分支中成功添加了留言回复功能，并将留言上限从20条提高到500条。

## 修改内容

### 1. 数据库迁移
- 创建了 `d1_migration.sql` 文件，包含Cloudflare D1兼容的SQL语句
- 添加了 `parent_id` 字段用于标识回复关系
- 添加了 `reply_count` 字段用于统计回复数量
- 创建了索引以提高查询性能

### 2. API后端 (`functions/api/messages.js`)
- **POST请求**：支持发布回复，检查留言上限（500条）
- **GET请求**：支持获取回复列表和统计信息
- **DELETE请求**：支持级联删除回复
- **统计功能**：区分主留言和回复的统计

### 3. 前端界面 (`comments/index.html`)
- **回复按钮**：每个留言添加回复按钮
- **回复表单**：支持回复特定留言
- **回复显示**：缩进显示回复内容
- **统计面板**：显示留言上限和回复总数
- **交互优化**：改进的用户体验

## 部署步骤

### 1. 数据库迁移
```bash
# 在Cloudflare Dashboard中执行以下SQL
npx wrangler d1 execute findorp-db --file=d1_migration.sql
```

### 2. 代码部署
```bash
# 切换到test分支
git checkout test

# 推送到GitHub
git push origin test

# 在Cloudflare Pages中部署test分支
```

## 功能特点

### 回复功能
- ✅ 支持对主留言进行回复
- ✅ 只允许一级回复（不能回复回复）
- ✅ 显示回复数量和内容
- ✅ 管理员可以管理所有回复

### 留言限制
- ✅ 主留言上限：500条
- ✅ 回复不计入上限
- ✅ 实时检查限制

### 兼容性
- ✅ Cloudflare D1数据库
- ✅ 渐进式增强（兼容旧数据）
- ✅ 响应式设计
